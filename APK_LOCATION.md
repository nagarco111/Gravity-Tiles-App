# APK Location Guide

After building the Gravity Tiles application, the APK files can typically be found in the following directory:

## For Android Studio Builds:
1. Open your project in Android Studio.
2. Navigate to:
   - `app/build/outputs/apk/`

   In this directory, you will find multiple folders corresponding to different build types (e.g., debug, release).

## Specific APK Locations:
- **Debug APK:** `app/build/outputs/apk/debug/`
- **Release APK:** `app/build/outputs/apk/release/`

These files can be easily accessed for testing or distribution.

Make sure to check the output of your build process to confirm the exact path where the APKs are generated in case of custom configurations.

Happy coding!
